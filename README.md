# biblioteca-combinat-ria

Contém biblioteca que visa auxiliar nos cálculos de Análise Combinatória 

Todas as funções contidas nesse repositório foram criados pelo autor João Víctor Costa de Oliveira (StormBreaker1726) utilizando conhecimentos adiquiridos nas disciplinas de Matemática Discreta, Cálculo I e II, Algoritmos, Estrutura de Dados, Laboratório de Programação I e II; também são baseados nas fórmulas contidas
no livro Análise Combinatória e Probabilidade (Augusto César Morgado).

Os códigos podem servir de base para estudantes que queiram desenvolver seus próprios, NUNCA PARA CÓPIA.

Os códigos originais tem as seguintes características:

1) Data de criação: 16/10/2020
2) Criador: João Víctor Costa de Oliveira (StormBreaker1726)
3) Data de Upload: 16/10/2020


Atualmente (16/10/2020) as funções presentes são:

1) permutsim(n): imprime o número de permutações simples que podem ser feitas como número n ( Pn )
2) combsim(p, n): imprime as combinações simples de n feitas de p a p ( C(n,p) )
3) desarranj(n): imprime o número de desarranjos (permutações caóticas) que podem ser feitas como número n ( Dn )
4) combwrep(n, p): imprime as combinações completas de n feitas de p a p ( C(n+p-1,p) )
