# biblioteca-combinat-ria

Biblioteca que visa fazer cálculos de Análise Combinatória a partir de funções criadas e implementadas.

Atualmente (16/10/2020) as funções presentes são:
--> permutsim(n): imprime o número de permutações simples que podem ser feitas como número n ( Pn )
--> combsim(p, n): imprime as combinações simples de n feitas de p a p ( C(n,p) )
--> desarranj(n): imprime o número de desarranjos (permutações caóticas) que podem ser feitas como número n ( Dn )
--> combwrep(n, p): imprime as combinações completas de n feitas de p a p ( C(n+p-1,p) )
